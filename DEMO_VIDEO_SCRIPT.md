## üé• Production Checklist

### Pre-Recording Setup

- [ ] **Clean WordPress Install**
  - Fresh WooCommerce store
  - Sample data loaded (orders, products, customers)
  - Plugin installed and activated
  - API key configured

- [ ] **Browser Setup**
  - Clean browser (no bookmarks bar, extensions visible)
  - Full screen mode
  - Zoom at 100%
  - Clear cache

- [ ] **Screen Recording**
  - Resolution: 1920x1080
  - Frame rate: 30fps
  - Audio: Good microphone
  - Test recording first

- [ ] **Prepare Questions**
  - Write 3-4 questions in advance
  - Test that they work
  - Know what responses to expect

- [ ] **Script Ready**
  - Voiceover script printed/on screen
  - Timing notes ready
  - CTA text prepared

### During Recording

- [ ] **Speak clearly and at moderate pace**
- [ ] **Pause between sections**
- [ ] **Let responses fully load before moving on**
- [ ] **Keep cursor movements smooth**
- [ ] **Don't rush - quality over speed**

### Post-Recording

- [ ] **Edit video**
  - Add intro/outro
  - Add text overlays
  - Add background music (optional)
  - Add captions/subtitles

- [ ] **Export**
  - Format: MP4
  - Resolution: 1080p
  - File size: Under 50MB (for WordPress.org)

- [ ] **Upload**
  - YouTube (for embedding)
  - Vimeo (alternative)
  - Self-hosted (if needed)
