# Video Production Guide - Step by Step

## üé¨ Complete Guide to Creating Your Demo Video

### Phase 1: Preparation (30-60 minutes)

#### 1.1 Set Up Your Demo Environment

**WordPress Setup:**
```bash
# Ensure you have:
- Clean WordPress install
- WooCommerce installed and activated
- Your plugin installed and activated
- API key configured
```

**Sample Data:**
- Create 20-30 sample orders (various dates, statuses)
- Create 10-15 products (different categories)
- Create 10-15 customers
- Add some coupons
- Create some refunds

**Test Questions:**
- Test all questions you'll ask in the video
- Ensure responses are good
- Note any delays or issues

#### 1.2 Prepare Your Screen

**Browser Setup:**
- Use Chrome or Firefox (clean, no extensions visible)
- Full screen mode (F11)
- Zoom at 100%
- Clear browser cache
- Close unnecessary tabs

**Desktop Setup:**
- Clean desktop (no personal files visible)
- Hide notifications
- Close other applications
- Set screen resolution to 1920x1080

#### 1.3 Script Preparation

- Print or have script visible on second screen
- Mark timing points
- Practice voiceover (if using)
- Prepare questions list

---

### Phase 2: Recording (15-30 minutes)

#### 2.1 Choose Your Tool

**Option A: Loom (Easiest)**
- Free account: loom.com
- Browser extension
- One-click recording
- Auto-uploads to cloud
- Built-in editing

**Option B: OBS Studio (Free, Advanced)**
- Download: obsproject.com
- More control
- Better quality
- Free and open source

**Option C: QuickTime (Mac)**
- Built-in on Mac
- File ‚Üí New Screen Recording
- Simple and free

**Option D: ScreenFlow (Mac, Paid)**
- Professional tool
- $169 one-time
- Best for advanced editing

#### 2.2 Recording Settings

**Resolution:** 1920x1080 (Full HD)
**Frame Rate:** 30fps
**Audio:** 
- Use good microphone (USB mic or headset)
- Test audio levels first
- Record in quiet environment

**File Format:** MP4 (H.264)

#### 2.3 Recording Process

**Step 1: Test Recording (2 minutes)**
- Record a quick test
- Check audio levels
- Check video quality
- Adjust settings if needed

**Step 2: Record Full Demo**
- Follow the script
- Don't worry about mistakes (you can edit)
- Record in one take if possible
- Pause between sections if needed

**Step 3: Record Multiple Takes**
- Record 2-3 versions
- Pick the best one
- Don't stress about perfection

---

### Phase 3: Editing (1-2 hours)

#### 3.1 Choose Your Editor

**Option A: CapCut (Free, Easy)**
- Free desktop app
- Simple interface
- Good for beginners
- Auto-captions

**Option B: DaVinci Resolve (Free, Professional)**
- Professional features
- Steep learning curve
- Free forever

**Option C: iMovie (Mac, Free)**
- Built-in on Mac
- Simple and intuitive
- Good for basic editing

**Option D: Canva (Free, Online)**
- Web-based
- Simple editing
- Good for quick videos

#### 3.2 Editing Steps

**Step 1: Import Video**
- Import your recording
- Create new project
- Set resolution to 1920x1080

**Step 2: Trim & Cut**
- Remove mistakes
- Trim beginning/end
- Cut out long pauses
- Keep best takes

**Step 3: Add Text Overlays**
- Add plugin name at start
- Add question text (if needed)
- Add feature highlights
- Add CTA at end

**Step 4: Add Highlights**
- Circle/arrow for important parts
- Zoom in on key moments
- Box around responses
- Highlight charts

**Step 5: Add Background Music (Optional)**
- Choose instrumental track
- Set volume to 20-30%
- Fade in/out
- Don't overpower voiceover

**Step 6: Add Captions (If Voiceover)**
- Auto-generate captions
- Review and fix errors
- Style captions (font, color, position)

**Step 7: Add Intro/Outro (Optional)**
- Simple logo animation
- Plugin name
- "Get it free" message

**Step 8: Export**
- Format: MP4
- Resolution: 1920x1080
- Quality: High
- File size: Under 50MB (compress if needed)

---

### Phase 4: Optimization (30 minutes)

#### 4.1 Compress Video (If Needed)

**If file is over 50MB:**

**Option A: HandBrake (Free)**
- Download: handbrake.fr
- Open video
- Preset: Fast 1080p30
- Reduce quality slightly
- Export

**Option B: Online Tools**
- CloudConvert
- FreeConvert
- Compress online

**Option C: YouTube Compression**
- Upload to YouTube
- Download compressed version
- YouTube auto-compresses

#### 4.2 Create Thumbnail

**Requirements:**
- 1280x720px
- Clear and eye-catching
- Shows plugin name
- Includes "Free" or "Demo"

**Tools:**
- Canva (free templates)
- Photoshop
- GIMP (free)

**Elements:**
- Plugin logo
- "Free Demo" text
- Screenshot of chat interface
- WooCommerce logo

## üí° Pro Tips

1. **Keep it simple** - Don't overcomplicate
2. **Show real usage** - Not perfect scenarios
3. **Test first** - Record test before full demo
4. **Multiple takes** - Record 2-3 versions
5. **Edit carefully** - Remove mistakes, keep best parts
6. **Add captions** - Many watch without sound
7. **Optimize for mobile** - Many watch on phones
8. **Include CTA** - Always end with clear call-to-action
